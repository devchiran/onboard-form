# Onboarding Form [React + TypeScript + Vite + Tailwind + zod + React-hook-Form + RTL]

## To run the app locally

- Install the dependencies

```js
npm install
```

- Start the app

```js
npm run dev
```

- Run test cases

```js
npm run test
```

- Run test cases in watch mode

```js
npm run test:watch
```

- Check test coverage

```js
npm run test:coverage
```
